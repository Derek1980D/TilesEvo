# TilesEvo

A tile-based resource management game where tiles are resources themselves. Build and expand your grid to create and manage your world.

🎮 **[Play Now](https://derek1980d.github.io/tilesEvo/)** - Live version on GitHub Pages

## Features

- **2D Grid System**: Dynamic tile-based grid with canvas rendering
- **Resource Management**: Tiles serve as resources for building and expanding space
- **Tap to Build**: Click/tap tiles to place them on the grid
- **Vanilla JavaScript**: Pure JS, no frameworks or external libraries

## Game Mechanics

- Start with an empty grid
- Click tiles to place them
- Tiles are resources—use them strategically to expand your building space
- Future: Resource gathering, crafting, and progression systems

## How to Play

1. **[Play Online](https://derek1980d.github.io/tilesEvo/)** or open `index.html` locally in your browser
2. Click/tap any empty grid cell to place a tile (costs 1 resource)
3. Right-click to remove tiles (refunds resources)
4. Build strategically with limited resources!

## Development

Built with vanilla JavaScript and HTML5 Canvas. No dependencies required.
